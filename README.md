# PDF Architect

A professional-grade utility to convert multiple images (JPG/PNG) into a single, formatted A4 PDF document.

## Key Features
* **100% Client-Side:** Files never leave your browser. Privacy by design.
* **A4 Standardization:** Automatic scaling and centering for professional printing.
* **Zero Dependencies:** No backend required. Works on any modern browser.

## How to Use
1. Open `index.html` in any browser.
2. Drag and drop your images.
3. Click 'Generate' to download your multi-page PDF.
